## ربات سوسانو — نسخه ریفکتور شده از ربات میرزا

ربات سوسانو یک بازطراحی کامل و بهینه‌سازی شده از ربات میرزا است که با هدف ارتقای تجربه کاربری و افزایش کارایی طراحی شده است.

<details>
  <summary> اصلاحات و بهبودها </summary>


#### ✅ بهینه‌سازی کلی — بهبود سرعت و عملکرد ربات

#### ✅ رنگ‌بندی دکمه‌ها — امکان رنگی کردن دکمه‌های ربات در سمت کاربر و ادمین

#### ✅ ایموجی پرمیوم — قابلیت اعمال ایموجی‌های اختصاصی (نیاز به اکانت پرمیوم برای مالک ربات)

#### ✅ انبار‌داری پیشرفته — بخش مدیریت انبار در پنل با امکانات:
• فروش دستی عمده
• پنل اضطراری

#### ✅ بازطراحی پنل وب — بهبود ظاهر و افزودن امکانات بیشتر بخش ادمین به پنل وب

#### ✅ بازطراحی کامل مینی‌اپ — تجربه کاربری بهتر و مدرن‌تر

#### ✅ کارت رنگی مشاهده حجم — نمایش حجم با طراحی جذاب

#### ✅ دکمه شیشه‌ای — بخش ادمین با استایل مدرن شیشه‌ای

#### ✅ سایر امکانات — بهبودها و اصلاحات تکمیلی


  

</details>
